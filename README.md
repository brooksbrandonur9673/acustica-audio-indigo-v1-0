## FAQ

### How do I get updates?
Use the latest download link on the project page, or turn on automatic updates if that option is available in your setup.

### Where are presets stored?
Preset storage depends on your configuration and export choices. Keep backups in a location you can restore from easily.

### What if the plugin is not loading in my DAW?
Check the selected plugin format, confirm compatibility mode if needed, and verify that your DAW is scanning the correct plugin paths.

### Can I move my settings to another machine?
Yes. If you export your presets and configuration files, you can transfer them to another Windows system and restore them there.

### What should I do before batch changes?
Make a backup first. That makes it easier to roll back settings if you want to compare different configurations later.
